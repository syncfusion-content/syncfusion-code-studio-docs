# What’s New in v2.0.1

# Improvements

## Dashboard - Filter Option

The **Dashboard Filter** allows users to refine and analyze dashboard metrics with precision. It allows you to drill down into data by chat session to analyze session-level usage.

- Use the filter options to refine dashboard data:
    - **Chat Session** – Once a user or team is selected, you can filter further by chat sessions to analyze session-level usage.

- The dashboard will refresh automatically to show metrics for the selected filter.

## Manage Team Owners:
The **Manage Team Owners** feature enables administrators to assign oversight responsibilities to selected team leaders. This functionality ensures that teams have designated leaders who can monitor activities, maintain structure, and support administrative operations across multiple teams. Team Owners gain enhanced visibility and accountability, allowing them to oversee several teams simultaneously and ensure proper management practices are followed.

### Purpose of Team Owners
- **Visibility:** They can see and track activity across multiple teams.
- **Control:** They can edit and manage across those teams and ensure teams are properly structured.
- **Accountability:** They act as the authority responsible for team-level decisions.

> **Note:** Team Owners are not limited to a single team. They can be assigned to multiple teams simultaneously, allowing them to provide wider
 with Admin-level access.

## Dark Mode Support:
- The **Dark Mode Support** feature in Code Studio Enterprise Server introduces a unified dark themed interface designed for developers and administrators who prefer working in low light or visually minimal environments. This theme applies consistently to Dashboard, User and Teams, Fallback Policy, Budget and Settings.

- Dark Mode can be enabled at any time, giving users flexibility to switch between light and dark themes based on their working style and preference, ensuring a modern and visually streamlined experience throughout Code Studio Enterprise Server.
